
# 📈 Yes Bank AI Analytics Dashboard

AI-Powered Financial Intelligence & Stock Prediction Platform built using Streamlit, Machine Learning, and Financial Analytics.

---

## 🚀 Project Overview

The Yes Bank AI Analytics Dashboard is an interactive financial analytics platform designed to analyze historical stock market data and predict future stock prices using Machine Learning models.

The dashboard provides:

- OHLC market analysis
- AI-based stock prediction
- Forecast simulations
- Financial sentiment analysis
- Executive summary insights
- PDF report generation

---

## 🎯 Project Objectives

- Analyze historical Yes Bank stock prices
- Visualize OHLC market trends
- Predict stock closing prices using ML algorithms
- Provide forecasting simulations
- Perform NLP-based sentiment analysis
- Deliver professional executive-level insights

## ▶️ How to Run the Project

### 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/YesBank-AI-Analytics.git
```

### 2️⃣ Open Project Folder

```bash
cd YesBank-AI-Analytics
```

### 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

### 4️⃣ Run Streamlit App

```bash
streamlit run app.py
```
